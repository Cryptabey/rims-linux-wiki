## Network Security

- [WireShark](https://www.wireshark.org/) - Network protocol analyzer
- [BetterCap](https://www.bettercap.org/) - Swiss Army knife for networking security
- [EtherApe](https://etherape.sourceforge.io/) - EtherApe is a graphical network monitor for Unix modeled after etherman
- [justniffer](https://onotelli.github.io/justniffer/) - Network protocol analyzer that captures network traffic and produces logs in a customized way
